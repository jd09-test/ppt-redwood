### CRITICAL RULES for the `content` Field Format

For any rich text content in the `content` field, it must be represented as a list of HTML-formatted strings, where each string is one line of text.
Each HTML string must follow **strict** formatting rules:

- **Plain Text**: If the line is plain text, it must be only text with out any tag. Example: `This is plain text.`
- **List Indentation**: If the paragraph is a list item or indented text, must be enclosed in a `<p>` tag and set the indentation level with the `level` attribute of `<p>`: level="0" is normal paragraph, level="1" for first-level list, etc. Example: `<p level="1">This is list item 1</p>`
- **Fragment Wrapping**: If the text has styles, it must be enclosed in a `<span>` tag. Example: `This is <span style="font-weight:bold;">bold text</span>.`
- **Forbidden Tags**: Absolutely do not use <b>, <i>, <u>, <a>, or any other HTML tags â€” all formatting must go through `<span>`
- **Available Styles**:

  - Italic: `This is <span style="font-style:italic;">italic text</span>.`
  - Underline: `This is <span style="text-decoration:underline;">underlined text</span>.`
  - Hyperlink: `This is <span data-link="https://example.com">Link text</span>.`
